How to Edit Your Home Page

  1. View your home page here: http://fllmathdroid.github.io/[your title]
  2. Go to: https://github.com/, log on with your username/password
  3. Search organization "fllmathdroid", then find project repository for "fllmathdroid.github.io"
  4. Find the folder with your title, go inside the folder, and modify the index.html
  5. Commit your changes.
  6. Go back to your home page to view your changed. Wait for a few minuted for Guthub server to refresh.

Project TrashIndex

  1. Go to: https://github.com/, log on with your username/password
  2. Search organization "fllmathdroid", then find project repository for "TrashIndex"
  3. To get familiar, you can browse the folders and files from website
  4. To contribute to the project, clone to your local computer.
  5. This project is written in Java using Android Development Tools. So please install Android studio on your computer.
  6. Recommend to use Intellij Android Studio, downloaded from here: http://developer.android.com/sdk/index.html
  7. Import the cloned project into Andriod Studio. Build it and test.
